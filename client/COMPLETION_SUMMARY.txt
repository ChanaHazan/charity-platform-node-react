🔌 CUSTOM HOOKS PROVIDED:

1️⃣  useGroups()
    → Fetches all groups with loading/error states

2️⃣  useSolicitors(groupId)
    → Fetches solicitors for a specific group

3️⃣  useDonors(solicitorId)
    → Fetches donors for a specific solicitor

4️⃣  useCampaignStats()
    → Fetches global campaign statistics

5️⃣  useCountdown(endDate)
    → Returns countdown string that updates every second

6️⃣  useCampaignContext()
    → Access global campaign state and makeDonation function

📡 API INTEGRATION:

Services Layer provides functions for:
  • groupService.fetchGroups() / getGroupById()
  • solicitorService.fetchSolicitors() / getSolicitorsByGroupId() / addSolicitor()
  • donorService.fetchDonors() / getDonorsBySolicitorId()
  • campaignService.fetchCampaignStats() / donate()

Axios configured with:
  • Base URL from VITE_API_URL environment variable
  • Request interceptors for auth tokens
  • Response interceptors for error handling
