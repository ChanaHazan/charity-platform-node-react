API INTEGRATION:
═════════════════════════════════════════════════════════════════════════════════

Backend Endpoints Required:

Groups:
  GET    /api/groups              → List all groups
  GET    /api/groups/:id          → Get single group
  GET    /api/groups/:id/solicitors → Get group's solicitors

Solicitors:
  GET    /api/solicitors          → List all solicitors
  POST   /api/solicitors          → Add new solicitor
  GET    /api/solicitors/:id/donors → Get solicitor's donors

Donors:
  GET    /api/donors              → List all donors

Donations & Campaign:
  POST   /api/donations           → Record donation
  GET    /api/campaign/stats      → Get campaign statistics

CUSTOM HOOKS PROVIDED:
═════════════════════════════════════════════════════════════════════════════════

1. useGroups() - Fetch all groups
2. useSolicitors(groupId) - Fetch solicitors for group
3. useDonors(solicitorId) - Fetch donors for solicitor
4. useCampaignStats() - Fetch campaign statistics
5. useCountdown(endDate) - Live countdown timer

═════════════════════════════════════════════════════════════════════════════════

Step 1: Install Dependencies
  $ cd client
  $ npm install

Step 2: Configure API URL
  Create client/.env.local:
  VITE_API_URL=http://localhost:5000/api

Step 3: Start Development Server
  $ npm run dev

Step 4: Open Browser
  http://localhost:5173

Build for Production:
  $ npm run build
